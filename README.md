# student-mental-health-analytics# 🧠 Student Mental Health Analytics

A data science project analyzing mental health patterns among university students and predicting depression risk.

## What I Built
- Exploratory Data Analysis (EDA) with 5 visualizations
- Depression risk classifier using Random Forest (71% accuracy)
- Interactive Streamlit web app for real-time prediction

## Key Findings
- Married students are most at risk of depression (highest feature importance at 0.31)
- Mid-range CGPA students (3.00-3.49) show highest depression rates
- Depression and anxiety have a moderate correlation (0.27)
- Year 1 and Year 3 students report highest anxiety levels

## Tech Stack
Python, Pandas, Scikit-learn, Matplotlib, Seaborn, Streamlit

## How to Run
pip install pandas scikit-learn matplotlib seaborn streamlit
streamlit run app.py
